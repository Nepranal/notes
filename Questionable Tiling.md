# Problem

> Is it possible to tile an 8 × 8 board with dominoes (2 × 1 tiles) so that no two dominoes form a 2 × 2 square?

# Solution

Consider the corner of the 8 x 8 board. It must be that the domino sticks to one the edges. If we jus take one of them, we would find that in order to not have any 2 x 2 squares, you'll have to draw the following
